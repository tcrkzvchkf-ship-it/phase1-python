#Phase 1 - Python Foundations

A 3 week python learning project covering core programming concepts data structures, OOP, File I/O, and data analysis

## Projects Built

### To-Do List CLI App
- Command line task manager with persistent JSON storage
- Built with Python, OOP, and file I/O
- Features: add, view, complete, and delete tasks

### Contact Book
- Python contact manager using nested dictionaries
- Features: add, find, and display contacts

### Number Guessing Game
- Interactive terminal game using while loops and conditional logic

### Titanic Data Analysis
- Explored real world dataset with 891 passengers
- Discovered 38.4% survival rate and gender survival disparity
- Built with NumPy and pandas

## Skills Learned
- Python fundamentals (variables, loops, functions, OOP)
- Git & GitHub (branches, commits, pull requests)
- File I/O and JSON data persistence
- HTML, CSS, JavaScript (interactive portfolio page)
- NumPy and pandas (data analysis)

## How to Run


### Data Analysis
